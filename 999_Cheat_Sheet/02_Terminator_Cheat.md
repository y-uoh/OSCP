| 操作 | キー |
| --- | --- |
| 水平に分割 | Ctrl + Shift + o |
| 垂直に分割 | Ctrl + Shift + e |
| 分割した一個分削除 | Ctrl + Shift + w |
| 分割した間の移動 | Ctrl + Tab （+Shiftで逆方向） |
| その分割した中で検索 | Ctrl + Shift + f |
| Next（検索） | Enter |
| 分割した一個の縦横幅を調節 | Ctrl + Shift + 矢印キー |
| 新規ウィンドウ | Ctrl + Shift + i |
| 新規タブ | Ctrl + Shift + t |
| 左のタブに移動 | Ctrl + PgUp |
| 右のタブに移動 | Ctrl + PgDn |
| 全ウィンドウを一気に削除 | Ctrl + Shift + q |
