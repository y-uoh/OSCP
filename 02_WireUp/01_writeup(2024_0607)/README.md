- ただやるだけで終わらせない
  - 競争するがあまり、よくわかんないのにとりあえず解くだけというのは避ける。
  - あくまで***自分のための勉強***であることを意識する
- 競争することは忘れない
  -　漫然としないように、緊張感を持つように
- ヒントは見ない
  - まずは時間いっぱいまでは足掻く
  - とにかく***調べて***、***試して***、何でよかったのか、何がダメだったのかを洗いだす。
- write upをしっかり書く
  - できたことより、考えたこと、`思考過程`を明確にする
  - 思考を整理して、無駄なことをそぎ落とす
  - 可能なら最初から再現できるかやってみる
